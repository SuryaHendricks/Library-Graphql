Technologies
----
*GraphQL*, *NodeJS*, *Express*, *MongoDB*, *JSW*

How to run the projet
----

1. Clone the repository.

2. Launch the terminal from the project's root folder and type **npm run dev**.

3. Navigate to **[localhost:5000/graphql]**.
